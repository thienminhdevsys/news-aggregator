This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Ngôn ngữ và Múi giờ

- **Ngôn ngữ**: Tiếng Việt
- **Múi giờ**: UTC+7 (Giờ Đông Dương)

## Dự án

Hệ thống tổng hợp tin tức tự động (News Aggregator) sử dụng n8n.

## Tính năng

- Thu thập tin tức từ RSS feeds
- Lọc theo từ khóa quan tâm
- Lưu trữ dữ liệu vào JSON
- Gửi báo cáo email hàng ngày
- Lên lịch tự động với Cron

## Công nghệ

- **n8n**: Automation platform
- **RSS Parser**: Đọc feeds
- **JSON Storage**: Lưu trữ
- **Gmail/SMTP**: Gửi email

## Quy tắc

- Không xóa file khi chưa hỏi
- Không commit git khi chưa được phép
- Backup dữ liệu trước khi thay đổi cấu trúc

## Hướng dẫn

Khi làm việc với người dùng trong thư mục này:
- Sử dụng tiếng Việt để giao tiếp
- Chuyển đổi thời gian sang múi giờ UTC+7 khi hiển thị
- Định dạng ngày tháng theo chuẩn Việt Nam (dd/mm/yyyy)
